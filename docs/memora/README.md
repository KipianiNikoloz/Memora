# Memora Knowledge Base

Memora is a personal reflection platform built around a library metaphor: every life deserves a library, every meaningful memory can become an entry, and revisiting those entries helps users notice growth that would otherwise stay invisible.

This knowledge base is the persistent product record for future implementation. It synthesizes:

- `static/Report.pdf`: project vision, problem, solution, AI librarian concept, use cases, and future potential.
- `static/ui/memora-ui1.png` through `static/ui/memora-ui10.png`: personas, journeys, information architecture, visual direction, components, product screens, states, and constraints.

## Decision Status

These docs define the intended product and MVP direction. They are not application code. Future implementation changes should treat these files as the source of truth unless a later OpenSpec change explicitly updates them.

## Product Thesis

People often compare themselves to others while overlooking their own progress. Memora helps users compare themselves to their past selves by turning meaningful moments into structured knowledge they can revisit, reflect on, and grow from.
