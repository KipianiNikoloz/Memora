## Why

Memora's main product surface is the personal library. Users need to browse, search, filter, and revisit saved memories through a shelf-based interface.

## What Changes

- Build the authenticated Library Explorer.
- Add shelves, entry cards, search, filters, sorting, empty state, add-entry action, and entry detail navigation.
- Use private memory-entry data only for the signed-in user.

## Capabilities

### New Capabilities

- `library-explorer`: Provides the main private library browsing and revisit experience.

### Modified Capabilities

- None.

## Impact

- Adds the primary authenticated app screen.
- Depends on memory-entry domain and new-entry flow.
