## Why

Memora handles sensitive emotional data, so users need visible privacy controls, editable preferences, and clear safety boundaries before the MVP is demo-ready.

## What Changes

- Add settings and privacy surfaces.
- Add AI tone default preference.
- Add data deletion/export expectations where feasible for MVP.
- Add privacy, terms, and wellness boundary copy.
- Add crisis-like input handling surfaces where AI is used.

## Capabilities

### New Capabilities

- `settings-privacy-safety`: Provides user settings, privacy controls, and safety boundary UX.

### Modified Capabilities

- None.

## Impact

- Adds settings UI and safety copy.
- Touches AI and data flows to respect user preferences and privacy expectations.
