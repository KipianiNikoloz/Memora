## Why

The AI librarian is Memora's differentiating reflective companion. It must support meaningful guidance while staying user-controlled, provider-agnostic, and non-clinical.

## What Changes

- Add an AI provider abstraction with a mock provider.
- Add tone modes: Motivational, Humorous, and Wise.
- Add title suggestions, entry reflections, revisit prompts, and insight summary primitives.
- Add safety boundaries for sensitive or crisis-like input.

## Capabilities

### New Capabilities

- `ai-librarian`: Provides provider-agnostic supportive AI reflection for Memora.

### Modified Capabilities

- None.

## Impact

- Adds AI service interfaces and UI integration points.
- Depends on private memory entries and core memory readiness.
