## Why

Memora should help users notice growth patterns across their private entries. The Insights dashboard turns saved memories into gentle, non-diagnostic reflection.

## What Changes

- Add authenticated Insights dashboard.
- Add mood trends, milestone highlights, revisit recommendations, and AI summary.
- Use private entries and AI librarian primitives.
- Keep insights supportive and non-score-like.

- None.
