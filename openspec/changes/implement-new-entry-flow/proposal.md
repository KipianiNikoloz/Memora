## Why

Capturing a meaningful moment is the first core Memora workflow. The MVP needs a low-friction New Entry flow that feels calm, private, and emotionally gentle.

## What Changes

- Build the New Entry screen and form.
- Add emotion chips, memory and lesson prompts, tags, life phase controls, save behavior, and validation states.
- Connect the form to the memory-entry domain.
- Prepare the post-save hook for AI title suggestions.

## Capabilities

### New Capabilities

- `new-entry-flow`: Provides the user-facing flow for capturing and saving a memory entry.

### Modified Capabilities

- None.

## Impact

- Adds the first complete authenticated product workflow.
- Depends on auth and memory-entry domain specs.
